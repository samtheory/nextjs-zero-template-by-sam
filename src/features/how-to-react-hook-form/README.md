# how-to-react-hook-form

**Purpose:** Forms with React Hook Form + Zod validation.

## What You'll Learn
- Setting up `useForm` with `zodResolver`
- `register`, `handleSubmit`, `formState.errors`
- Using `Controller` for custom/third-party inputs
- `setError` for server-side error mapping
- `watch`, `setValue`, `reset`
- Multi-step form example

## Files
```
validators/
  login.schema.ts
  signup.schema.ts
  profile.schema.ts
components/
  LoginForm.tsx      — basic form with field errors
  SignupForm.tsx     — with password confirm refine
  ProfileForm.tsx    — with Controller example
```
