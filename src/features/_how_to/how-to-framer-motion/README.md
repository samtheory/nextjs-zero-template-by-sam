# how-to-framer-motion

**Purpose:** Animation patterns with Framer Motion (motion.dev) v12.

## What You'll Learn
- `motion.*` elements — `initial`, `animate`, `exit`, `transition`
- Variants for coordinated / staggered animations
- `AnimatePresence` for enter/exit animations
- Gesture animations: `whileHover`, `whileTap`, `drag`
- `useMotionValue`, `useTransform`, `useSpring` for scroll/mouse effects
- `layout` prop for automatic layout animations
- Page transitions with `template.tsx`

## Files
```
components/
  FadeInDemo.tsx          — basic initial/animate
  StaggeredList.tsx       — variants with staggerChildren
  ModalDemo.tsx           — AnimatePresence enter/exit
  GestureDemo.tsx         — whileHover, whileTap, drag
  ParallaxCard.tsx        — useMotionValue + useTransform
  LayoutAnimationDemo.tsx — layout prop
```
