# how-to-typescript

**Purpose:** Practical TypeScript patterns you'll use in this project every day.

## What You'll Learn
- Utility types: `Partial`, `Required`, `Pick`, `Omit`, `Record`, `ReturnType`, `Awaited`
- `Extract`, `Exclude`, `NonNullable`
- Generics with constraints
- Type guards and narrowing
- Discriminated unions
- Template literal types
- `as const` and `satisfies`

## Files
```
models/
  utility-types.example.ts    — all utility types with examples
  generics.example.ts         — generic patterns
  narrowing.example.ts        — type guards and discriminated unions
  advanced.example.ts         — template literals, as const, satisfies
```
