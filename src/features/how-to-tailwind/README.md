# how-to-tailwind

**Purpose:** Tailwind v4 CSS variable theming system.

## What You'll Learn
- `@theme` vs `@theme inline` — when to use each
- Defining color scales accessible as Tailwind utilities
- Semantic tokens that respond to dark mode
- Adding custom fonts via `next/font` + `@theme inline`
- Custom spacing, radius, and shadow tokens
- `dark:` prefix still works for explicit overrides

## Files
```
components/
  ColorPaletteDisplay.tsx    — visual display of all color tokens
  DarkModeDemo.tsx           — semantic vs raw token comparison
  TypographyDemo.tsx         — custom font display
  SpacingDemo.tsx            — custom spacing tokens
```
